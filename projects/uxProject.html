Portfolio Project


This project showcases the steps taken to create a responsive portfolio website. As the lead designer, I utilized Figma to design the user interface and create wireframes. Through a comprehensive case study, I conducted user research and iterated on the design to ensure a seamless user experience.

Portfolio project, sketches.

Low-fidelity, wireframes.

Case Study

Key Findings and Insights:

  • Users valued a clean and intuitive interface.
  • Mobile responsiveness was a top priority.
  • Clear navigation and well-organized content were essential.


Challenges and Solutions:

  • Balancing aesthetics and functionality.
  • Optimizing the design for different screen sizes.
  • Ensuring accessibility for all users.
